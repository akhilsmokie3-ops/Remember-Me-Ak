* **No API Keys.** No OpenAI bill. No network requests.

A hierarchical memory projection engine that outperforms standard Vector DBs.

* **Feature:** It "sleeps" to compress redundant vectors, preventing the context window from bloating while retaining infinite recall.

## 🚀 INSTALLATION (30 Seconds)
 
We built this for everyone. You do not need to be a Python wizard.
 
**Prerequisites:** Windows 10/11 (Linux support coming soon). 8GB RAM.
 
1. **Clone the Repo:**
 
    ```bash
    git clone https://github.com/merchantmoh-debug/Remember-Me-AI.git
    cd Remember-Me-AI
    ```
 
2. **Install the Brain (Zero Config):**
 
    ```bash
    python install_brain.py
    ```
 
    * *Automatically detects hardware and downloads DeepSeek-R1 (4.9 GB).*
    * *Verifies integrity via SHA-256.*
 
3. **Ignite the System:**
    Double-click **`start_sovereign.bat`**.
    * *Launches the backend engine.*
    * *Opens the "Cognitive Interface" in your browser.*

### 🔧 Build from Source (Optional)
*For Sovereign Engineers who do not trust pre-compiled binaries.*

Remember-Me bundles a pre-compiled `llama-server.exe` for accessibility. To build your own:
1.  Clone [llama.cpp](https://github.com/ggerganov/llama.cpp).
2.  Build the `server` target (`make server` or `cmake --build . --target server`).
3.  Replace `src/brain_engine/llama-server.exe` with your fresh binary.
